Spotify Clone

This project is a web page inspired by the popular music streaming service, Spotify. It features a clean and user-friendly interface that allows users to browse and listen to their favorite music. The page includes a navigation bar, a song list with album covers, a progress bar for tracking the playback, and control icons for play, pause, shuffle, and repeat. The project showcases a visually appealing design and demonstrates skills in web development and design. It provides a familiar and enjoyable music streaming experience for users.

Online_hunger

This project is a simple HTML webpage for an online food delivery service called "Online_hunger.com." The webpage includes various sections, such as a navigation bar, a welcome video, information about the services offered, a section showcasing client logos, a contact form, and a footer. Online Attendence System

This project is an Online Attendance System designed to simplify and streamline the process of recording and managing attendance for educational institutions, businesses, or any organization that requires attendance tracking. It offers a user-friendly interface that allows users to log in, mark attendance, and generate reports. Key features of the system include user authentication for secure access, the ability to create and manage courses or sessions, marking attendance for students or employees, and generating detailed attendance reports. The project demonstrates proficiency in web development, database management, and user authentication, providing a valuable tool for organizations seeking to digitize and automate their attendance tracking processes.
